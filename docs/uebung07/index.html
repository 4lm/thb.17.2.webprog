XPath Expressions

Datei: books.xml

Link: XPath Tester

Anweisung: XPath Expression:
The Book element * oder /Book
All Titles within Book */title oder /Book/title
All Titles //title
All Titles within Person //Person/title besser /Book/Person/title (// ist teuer)
All Elements within Person //Person/* besser /Book/Person/*
All Elements within the second Person //Person[2]/* besser /Book/Person[2]/*
All Type Attributes //@type besser /Book/title/@type
All Title Elements with at least one attribute //title[@*] besser /Book/title[@*]
All German Book titles //title[@lang='de'] besser /Book/title[@lang='de']
All Elements with exactly two Person elements *[count(Person) = 2] (eq 2 auch möglich)
All Elements with exactly three child elements */*[count(*) = 3]
All First and last names /Book/Person/*[self::LastName or self::FirstName]
All ancestors of LastName /Book/Person/LastName/ancestor::*
All person ancestors of Title /Book/Person/title/ancestor::Person
All sibling elements following FirstName /Book/Person/FirstName/following-sibling::* oder n Person: (/Book/Person/FirstName)[n]/following-sibling::*
All Role elements 'Editor' /Book/Person/role[text()='Editor']
All Titles containing the string "rie" /Book/title[text()='rie']
All LastNames with FirstName="John" /Book/title[contains(text(), 'rie')]
All Persons with a title element /Book/Person/title/ancestor::Person besser Book/Person[title]
