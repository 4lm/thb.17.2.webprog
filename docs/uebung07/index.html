XPath Expressions

Datei: books.xml

Link: XPath Tester

  1. The Book element: * oder /Book
  2. All Titles within Book: */title oder /Book/title
  3. All Titles: //title
  4. All Titles within Person: //Person/title besser /Book/Person/title (// ist teuer)
  5. All Elements within Person: //Person besser /Book/Person
  6. All Elements within the second Person: //Person[2] besser /Book/Person[2]
  7. All Type Attributes: //@type besser /Book/title/@type
  8. All Title Elements with at least one attribute: //title[@*] besser /Book/title[@*]
  9. All German Book titles: //title[@lang='de'] besser /Book/title[@lang='de']
  10. All Elements with exactly two Person elements: *[count(Person) = 2] (eq 2 auch möglich)
  11. All Elements with exactly three child elements: */*[count(*) = 3]
  12. All First and last names: /Book/Person/*[self::LastName or self::FirstName]
  13. All ancestors of LastName: /Book/Person/LastName/ancestor::*
  14. All person ancestors of Title: /Book/Person/title/ancestor::Person
  15. All sibling elements following FirstName: /Book/Person/FirstName/following-sibling::* oder 1 Person: (/Book/Person/FirstName)[1]/following-sibling::* oder 2 Person: (/Book/Person/FirstName)[2]/following-sibling::*
  16. All Role elements 'Editor': /Book/Person/role[text()='Editor']
  17. All Titles containing the string "rie": /Book/title[text()='rie']
  18. All LastNames with FirstName="John": /Book/title[contains(text(), 'rie')]
  19. All Persons with a title element: /Book/Person/title/ancestor::Person besser Book/Person[title]
